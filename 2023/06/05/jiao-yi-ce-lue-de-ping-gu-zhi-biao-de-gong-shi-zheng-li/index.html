2 收益指标

2.1 累计收益率 Cumulative Returns

单期收益率：
$$
R_t = \frac{P_t - P_{t-1}}{P_{t-1}}
$$
其中，$R_t$ 是第 $t$ 期的单期简单收益率，$P_t$ 是第 $t$ 期的价格，$P_{t-1}$ 是第 $t-1$ 期的价格。

import empyrical

def simple_returns(prices)：
    ...
    out = np.diff(prices, axis=0)
    np.divide(out, prices[:-1], out=out)
    ...

累计收益率：
$$
R_t(k) = \prod_{j=0}^{k-1}(1+R_{t-j})-1
$$
其中，$R_t(k)$ 是第 $t$ 期的 $k$ 期简单收益率，当 $k$ 等于 $t$ 时，它被称作累计收益率，即 $R_t(t)$ 。

import empyrical

def cum_returns(returns, starting_value=0, out=None):
    ...
    np.add(returns, 1, out=out)
    out.cumprod(axis=0, out=out)
    ...

def cum_returns_final(returns, starting_value=0):
    ...
    result = np.nanprod(returns + 1, axis=0)
    ...

2.2 年化收益率 Annualized Returns

1 含义：在一段时间内的收益，假定一年都是这个水平，进而折算的年收益率。

2 公式：
$$
AR = (1+R_t(t))^{1/(T/t)}-1
$$
其中，$R_t(t)$ 是累计收益率，$T$ 是整个时期的单位长度，一年有252的天数可进行交易。

3 源码：

import empyrical

def annual_return(returns, period=DAILY, annualization=None):
    ...
    ann_factor = annualization_factor(period, annualization)
    num_years = len(returns) / ann_factor
    ending_value = cum_returns_final(returns, starting_value=1)

    return ending_value ** (1 / num_years) - 1

2.3 损益 Profit and Loss

1 含义：衡量交易利润与交易损失之间的比例。

2 公式：当天持仓量 * 价格变动

2.4 胜率 Win Ratio

1 含义：赢钱次数除以总买入次数。

2 源码：

import quantstats

def win_rate(returns, aggregate=None, compounded=True, prepare_returns=True):
    ...
    return _win_rate(returns)


def _win_rate(series):
    ...
    return len(series[series > 0]) / len(series[series != 0])

3 风险指标

3.1 阿尔法 Alpha

1 含义：与市场波动无关的回报，也叫超额收益。 比如投资者获得了15%的回报，其基准获得了10%的回报，那么Alpha或者价值增值的部分就是5%

2 公式：
$$
Alpha = R - [R_f + \beta * (R_m - R_f)]
$$
其中，$R$ 是策略收益率，$R_f$ 是无风险利率，$\beta$ 是贝塔，$R_m$ 是基准收益率。

3 源码：

import empyrical

def alpha(returns,
          factor_returns,
          risk_free=0.0,
          period=DAILY,
          annualization=None,
          out=None,
          _beta=None):
    ...
    return alpha_aligned(
        returns,
        factor_returns,
        risk_free=risk_free,
        period=period,
        annualization=annualization,
        out=out,
        _beta=_beta
    )


def alpha_aligned(returns,
                  factor_returns,
                  risk_free=0.0,
                  period=DAILY,
                  annualization=None,
                  out=None,
                  _beta=None):
    ...
    adj_returns = _adjust_returns(returns, risk_free)
    adj_factor_returns = _adjust_returns(factor_returns, risk_free)
    alpha_series = adj_returns - (_beta * adj_factor_returns)

    out = np.subtract(
        np.power(
            np.add(
                nanmean(alpha_series, axis=0, out=out),
                1,
                out=out
            ),
            ann_factor,
            out=out
        ),
        1,
        out=out
    )
    ...

3.2 贝塔 Beta

1 含义：策略对大盘变化的敏感性。例如一个策略的Beta为1.5，则大盘涨1%的时候，策略可能涨1.5%

2 公式：
$$
beta = \frac{Cov(R_e,R_m)}{Var(R_m)}
$$
其中，$R_e$ 是策略收益率，$R_m$ 是基准收益率。

3 源码：

import empyrical

def beta(returns, factor_returns, risk_free=0.0, out=None):
    ...
    return beta_aligned(
        returns,
        factor_returns,
        risk_free=risk_free,
        out=out,
    )


def beta_aligned(returns, factor_returns, risk_free=0.0, out=None):
    ...
    independent = np.where(
        isnan(returns),
        nan,
        factor_returns,
    )
    ind_residual = independent - nanmean(independent, axis=0)

    covariances = nanmean(ind_residual * returns, axis=0)

    np.square(ind_residual, out=ind_residual)
    independent_variances = nanmean(ind_residual, axis=0)
    independent_variances[independent_variances < 1.0e-30] = np.nan

    np.divide(covariances, independent_variances, out=out)
    ...

3.3 年化波动率 Annual Volatility

1 含义：策略收益的波动程度，是对资产收益率不确定性的衡量，用于反映金融资产的风险水平。

2 公式：
$$
AV = \sigma_p * (T^{1/2})
$$
$$
\sigma_p = \sqrt{\frac{1}{n}\sum_{i=1}^n(r_p- \frac{1}{n}\sum_{i=1}^nr_p)^2 }
$$
其中，$\sigma_p$ 是策略收益率的波动率，$T$ 是整个时期的单位长度，一年有252的天数可进行交易，$r_p$ 是策略收益率。

3 源码：

import empyrical

def annual_volatility(returns,
                      period=DAILY,
                      alpha=2.0,
                      annualization=None,
                      out=None):
    ...
    ann_factor = annualization_factor(period, annualization)
    nanstd(returns, ddof=1, axis=0, out=out)
    out = np.multiply(out, ann_factor ** (1.0 / alpha), out=out)
    ...

3.4 年化下行波动率 Annual Downside Risk

1 含义：相比于普通波动率，下行波动率对收益率的波动方向进行了区分，只有当收益率向下波动时才视为风险。

2 公式：
$$
ADR = DR (T^{1/2})
$$
$$
DR = \sqrt{\frac{1}{n}\sum_{i=1}^n(R_p(i)- R_m(i))^2f(i) }
$$
如果 $R_p(i) < R_m(i)$ ，则 $f(i)=1$ ; 如果 $R_p(i) \geq R_m(i)$ ，则 $f(i)=0$ 。

其中，$DR$ 是下行波动率，$T$ 是整个时期的单位长度，如果是日频则一年有252天可进行交易，$R_p(i)$ 是第 $i$ 期的策略收益率，$R_m(i)$ 是第 $i$ 期的基准收益率。

3 源码：

import empyrical

def downside_risk(returns,
                  required_return=0,
                  period=DAILY,
                  annualization=None,
                  out=None):
    ...
    downside_diff = np.clip(
        _adjust_returns(
            np.asanyarray(returns),
            np.asanyarray(required_return),
        ),
        np.NINF,
        0,
    )

    np.square(downside_diff, out=downside_diff)
    nanmean(downside_diff, axis=0, out=out)
    np.sqrt(out, out=out)
    np.multiply(out, np.sqrt(ann_factor), out=out)
    ...

3.5 信息比率 Information Ratio

1 含义：信息比率与夏普比率相似，主要区别在于夏普比率使用无风险收益作为基准，而信息比率使用风险指数作为基准。

2 公式：
$$
IR = \frac{E[R_p - R_b]}{\sqrt{var[R_p - R_b]}}
$$
其中，$R_p$ 是策略收益率，$R_b$ 是基准收益率。

3 源码：

import quantstats

def information_ratio(returns, benchmark, prepare_returns=True):
    ...
    diff_rets = returns - _utils._prepare_benchmark(benchmark, returns.index)

    return diff_rets.mean() / diff_rets.std()

3.6 年化夏普比率 Annual Sharpe Ratio

1 含义：资产收益对投资者所承担风险的补偿程度。当以一个相同基准来比较两种资产之时，夏普比率较高的资产在相同风险下收益更好。

2 公式：
$$
ASR = SR * (T^{1/2})
$$
$$
SR = \frac{E[R_a - R_b]}{\sqrt{var[R_a-R_b]}}
$$
其中，$SR$ 是夏普比率，$R_a$ 是策略收益率，$R_b$ 是无风险利率。

3 源码：

import empyrical

def sharpe_ratio(returns,
                 risk_free=0,
                 period=DAILY,
                 annualization=None,
                 out=None):
    ...
    np.multiply(
        np.divide(
            nanmean(returns_risk_adj, axis=0),
            nanstd(returns_risk_adj, ddof=1, axis=0),
            out=out,
        ),
        np.sqrt(ann_factor),
        out=out,
    )
    ...

3.7 索提诺比率 Sortino Ratio

1 含义：与夏普比率类似，所不同的是它使用了下行波动率，即策略的正回报率符合投资人的需求，不应计入风险调整。

2 公式：
$$
SR = \frac{R_p - r_f}{\sigma_{d}}
$$
其中，$R_p$ 是策略年化收益率，$r_f$ 是无风险利率，$\sigma_{d}$ 是下行波动率。

3 源码：

import empyrical

def sortino_ratio(returns,
                  required_return=0,
                  period=DAILY,
                  annualization=None,
                  out=None,
                  _downside_risk=None):
    ...
    average_annual_return = nanmean(adj_returns, axis=0) * ann_factor
    annualized_downside_risk = (
        _downside_risk
        if _downside_risk is not None else
        downside_risk(returns, required_return, period, annualization)
    )
    np.divide(average_annual_return, annualized_downside_risk, out=out)
    ...

3.8 卡玛比率 Calmar Ratio

1 含义：夏普比率的另一个变种，同样是用来衡量策略的风险收益比。相比于前者，卡玛比率使用最大回撤作为风险，同时没有考虑无风险利率。

2 公式：
$$
CR = \frac{annual \; return}{max \; drawdown}
$$
其中，$annual \; return$ 是策略年化收益率，$max \; drawdown$ 是最大回撤。

3 源码：

import empyrical

def calmar_ratio(returns, period=DAILY, annualization=None):
    ...
    max_dd = max_drawdown(returns=returns)
    if max_dd < 0:
        temp = annual_return(
            returns=returns,
            period=period,
            annualization=annualization
        ) / abs(max_dd)
    ...

3.9 最大回撤 Max DrawDown

1 含义：策略可能出现的最糟糕的情况，是指在某一段时期内累计收益率从最高点开始回落到最低点的幅度。

2 公式：
$$
MDD(T) = max_{\tau \in (0,T)}[max_{t \in (0, \tau)}X(t)-X(\tau)]
$$
其中，$X(t)$ 是第 $t$ 期的累计收益率。

3 源码：

import empyrical

def max_drawdown(returns, out=None):
    ...
    cumulative = np.empty(
        (returns.shape[0] + 1,) + returns.shape[1:],
        dtype='float64',
    )
    cumulative[0] = start = 100
    cum_returns(returns_array, starting_value=start, out=cumulative[1:])

    max_return = np.fmax.accumulate(cumulative, axis=0)

    nanmin((cumulative - max_return) / max_return, axis=0, out=out)
    ...

3.10 最大回撤持续期 Max Drawdown Duration

1 含义：累计收益率从回撤开始到再创新高所经历的单位时间长度。相比于最大回撤，它用于刻画收益跌挫的时间维度。

3.11 跟踪误差 Tracking Error

1 含义：衡量策略收益和市场基准收益之间的差异。跟踪误差越大，表明策略偏离基准的程度越大。

3.12 胜负比 Win Loss Ratio

1 含义：赢的交易总数与输的交易总数之比。它侧重于描述赢的次数而非收益的多少。

2 公式：
$$
WLR = \frac{wins}{losses}
$$
其中，$wins$ 是收益为正的交易次数，$losses$ 是收益为负的交易次数。

3 源码：

import quantstats

def win_loss_ratio(returns, prepare_returns=True):
    return payoff_ratio(returns, prepare_returns)


def payoff_ratio(returns, prepare_returns=True):
    ...
    return avg_win(returns) / abs(avg_loss(returns))

3.13 盈亏比 Risk Return Ratio

1 含义： 每次交易的盈利和亏损的比例，代表投资的风险收益比。比如平均每次盈利30%，平均每次亏损10%，盈亏比就是3倍。

2 公式：
$$
RRR = \frac{E(R)}{\sqrt{var(R)}}
$$
或者
$$
RRR = \frac{R}{MDD}
$$
其中，$R$ 是策略收益率，$MDD$ 是最大回撤。

3 源码：

import quantstats

def risk_return_ratio(returns, prepare_returns=True):
    ...
    return returns.mean() / returns.std()

3.14 偏度 Skew

1 含义：描述数据分布的对称性，正态分布的偏度为0。计算数据样本的偏度，当偏度<0时，称为负偏，数据出现左侧长尾；当偏度>0时，称为正偏，数据出现右侧长尾。

2 公式：
$$
Skew[X] = \frac{E[(X-u)^3]}{(E[(X-u)^2])^{3/2}} = \frac{u_3}{\sigma^3}
$$
其中，$u$ 是随机变量 $X$ 的均值，$\sigma$ 是随机变量 $X$ 的标准差，$u_3$ 是 $X$ 的3阶中心矩。

3 源码：

import quantstats

def skew(returns, prepare_returns=True):
    ...
    return returns.skew()

3.15 峰度 Kurtosis

1 含义：描述数据分布陡峭或是平滑的情况。正态分布的峰度为3，峰度越大，代表分布越陡峭，尾部越厚；峰度越小，分布越平滑。

2 公式：
$$
Kurt[X] = \frac{E[(X-u)^4]}{(E[(X-u)^2])^2} = \frac{u_4}{\sigma^4}
$$
其中，$u$ 是随机变量 $X$ 的均值，$\sigma$ 是随机变量 $X$ 的标准差，$u_4$ 是 $X$ 的4阶中心矩。

3 源码：

import quantstats

def kurtosis(returns, prepare_returns=True):
    ...
    return returns.kurtosis()
