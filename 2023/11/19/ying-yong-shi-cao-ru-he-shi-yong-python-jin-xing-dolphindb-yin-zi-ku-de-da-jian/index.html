应用实操｜如何使用 Python 进行 DolphinDB 因子库的搭建

0 前言

DolphinDB 是高性能、分布式的时序数据库，以列存储、多语言支持、实时分析为特点，提供丰富的内置函数和易于部署的管理工具，适用于处理大规模数据和复杂分析任务。

本文选择 DolphinDB 作为金融数据库，同时先从数据库创建的角度进行讲解，再进一步介绍对应的数据表的持久化操作。

1 基于分区的因子库搭建

1.1 因子的计算

下面的例子是 “101 Formulaic Alphas” by Kakushadze (2015) 中的 98 号因子，给出了对应的计算公式与调用 DolphinDB 的 Python 代码。

# Alpha98: (rank(decay_linear(correlation(vwap, sum(adv5, 26.4719), 4.58418), 7.18088)) 
#        - rank(decay_linear(Ts_Rank(Ts_ArgMin(correlation(rank(open), rank(adv15), 20.8187), 8.62571), 6.95668), 8.07206)))

import dolphindb as ddb
import dolphindb.settings as keys

s = ddb.session()
s.connect("localhost", 8848, "admin", "123456")

def alpha98(t):
    t1 = s.table(data=t)
    t1.contextby(["date"]).update(cols=["rank_open","rank_adv15"], vals=["rank(open)","rank(adv15)"]).execute()
    t1.contextby(["PERMNO"]).update(["decay7", "decay8"], ["mavg(mcorr(vwap, msum(adv5, 26), 5), 1..7)",\
               "mavg(mrank(9 - mimin(mcorr(rank_open, rank_adv15, 21), 9), true, 7), 1..8)"]).execute()
    t1 = s.table(data=t)
    return t1.select("PERMNO, date,decay7, decay8, rank(decay7)-rank(decay8) as A98")\
        .contextby(["date"])\
        .executeAs("alpha98")

US = s.loadTable(tableName="US", dbPath="dfs://US").executeAs("US")
result = alpha98(US.tableName())
s.close()

1.2 数据库创建

这里简单以日频因子存储库表为例，由于 DolphinDB 官方给出了相关的配置资料，这里我们直接套用它的配置参数进行搭建，即存储结构采用窄表、存储使用时间年和因子名进行分区、分区内部使用标的ID和时间戳进行排序。

下面给出了对应的 DolphinDB 脚本代码，它需要在数据库命令行执行。

dbName, tbName = "dfs://dayFactorDB", "dayFactorTB"

db1 = database(directory="", partitionType=RANGE, partitionScheme=date(datetimeAdd(1980.01M,seq(0,80)*12,'M')))
db2 = database(directory="", partitionType=VALUE, partitionScheme=["alpha1","alpha2"])
db = database(directory=dbName, partitionType=COMPO, partitionScheme=[db1, db2], engine='TSDB', atomic='CHUNK')


schemaTB = table(array(DATE, 0) as tradetime, 
                array(SYMBOL, 0) as securityid, 
                array(SYMBOL, 0) as factorname, 
                array(DOUBLE, 0) as value)

db.createPartitionedTable(table=schemaTB,
                          tableName=tbName,
                          partitionColumns=["tradetime", "factorname"],
                          compressMethods={"tradetime":"delta"}, 
                          sortColumns=["securityid", "tradetime"],
                          keepDuplicates=ALL,
                          sortKeyMappingFunction=[hashBucket{,500}])

为了验证上述脚本的有效性，这里进一步对得到的分区表进行数据添加，并进行查询。

tb = loadTable(tableName=tbName, database=dbName)                          
testDict = table([2021.05.07,2021.05.06,2021.05.07,2021.05.06] as tradetime,

                 ['AAPL','AMZN','AMZN','AAPL'] as factorname,

2 分区数据表的基本操作

在 DolphinDB 中一共存在内存表和分区表两种表结构，考虑到数据库最基本的持久化功能，这里只选择分区表的操作进行介绍。

在这些操作的代码执行之前，需要事先完成数据库表的搭建，并添加如下代码在开头。

import dolphindb as ddb

s = ddb.session()
s.connect("43.142.109.196", 8848, "admin", "123456")

dbName, tbName = "dfs://dayFactorDB", "dayFactorTB"

2.1 加载

# 直接加载
trade = s.loadTable(tableName=tbName, dbPath=dbName).toDF()

# 加载同时过滤
trade = s.loadTableBySQL(tableName=tbName, dbPath=dbName,
                         sql=f"select * from {tbName} where tradetime>2021.05.06").toDF()

2.2 添加

pool = ddb.DBConnectionPool("43.142.109.196", 8848, 20, "admin", "123456")
appender = ddb.PartitionedTableAppender(dbName, tbName, "tradetime", pool)

data = pd.DataFrame({'tradetime': np.array(['2019-03-03', '2019-03-04', '2019-03-05', '2019-03-03'], dtype="datetime64[D]"),
         'securityid': ['sh30000', 'sh30003', 'sh30002', 'sh30003'],
         'factorname': ['AAPL', 'AMZN', 'AMZN', 'AAPL'],
         'value': [130.21, 3306.37, 3291.61, 129.74]})
re = appender.append(data)

2.3 更新

tb = s.loadTable(tableName=tbName, dbPath=dbName)
tb.update(["value"], ["130.05"]).where("securityid=`sh30000").execute()

2.4 删除

# 删除表记录
tb = s.loadTable(tableName=tbName, dbPath=dbName)
tb.delete().where("securityid=`sh30000").execute()

# 删除表
s.dropTable(tableName=tbName, dbPath=dbName)

2.5 查询

tb = s.loadTable(tableName=tbName, dbPath=dbName)
trade = tb.select('avg(value)').groupby(['factorname']).having('count(securityid)>1').toDF()
